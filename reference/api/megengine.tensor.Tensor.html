megengine.tensor.Tensor¶

class Tensor(data, dtype=None, device=None, is_const=False, no_cache=False, name=None)[源代码]¶

基类：Tensor, megengine.core.tensor.array_method.ArrayMethodMixin

A tensor object represents a multidimensional, homogeneous array of fixed-size items.

参数

  • data (Union[ForwardRef, ndarray, list, ForwardRef]) – The value of returned Tensor.

  • dtype (dtype) – The dtype of returned Tensor. Uses data’s dtype if not specified.

  • device (str) – The desired device of returned Tensor. Uses get_default_device if not specified.

  • is_const (bool) – Whether make it a ImutableTensor in tracing mode.

  • no_cache (bool) – Whether cache it for memory sharing.

  • name (str) – Used to improve convenience in graph operation on dumped model.
