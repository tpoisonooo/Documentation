NodeFilter¶

NodeFilter

Filter on node iterator.

NodeFilter.type

filter by specific node type

NodeFilter.check_type

assert that all oprs produced by this iterator are instances of certain type

NodeFilter.not_type

remove oprs of specific type

NodeFilter.param_provider

get ParamProvider oprs; shorthand for .type(ParamProvider)

NodeFilter.data_provider

get DataProvider oprs; shorthand for .type(DataProvider)

NodeFilter.name

filter by node name

NodeFilter.has_input

an opr is kept if it has given var as one of its inputs

NodeFilter.as_list

consume this iterator and return its content as a list

NodeFilter.as_unique

assert that this iterator yields only one node and return it

NodeFilter.as_dict

construct an ordered dict to map from node names to objects in this iterator

NodeFilter.as_count

consume this iterator and get the number of elements
